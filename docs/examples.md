# Examples & supported diagram types

The repository carries two parallel example sets:

- **[`examples/gen2/remote_testing/`](https://github.com/doubleSlashde/umltheme/tree/main/examples/gen2/remote_testing)** — `!include` via **GitHub raw** URLs
- **[`examples/gen2/local_testing/`](https://github.com/doubleSlashde/umltheme/tree/main/examples/gen2/local_testing)** — **relative** includes into [`doubleslash/`](https://github.com/doubleSlashde/umltheme/tree/main/doubleslash)

Diagram catalogues with narration also live under [`examples/input/`](https://github.com/doubleSlashde/umltheme/tree/main/examples/input).
