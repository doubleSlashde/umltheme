# doubleSlash plantUML Theme

This repository hosts **doubleSlash PlantUML styling** theme files that you reference with `!include` at the top of your diagrams.

## Gen2 highlights

| Feature                  | Description                                                                                                            |
| ------------------------ | ---------------------------------------------------------------------------------------------------------------------- |
| **Light & dark mode**    | Use dedicated light/dark themes or the universal bundle                                                                |
| **Styling updates**      | Refined palette and clearer diagram defaults                                                                           |
| **Simpler structure**    | Consolidated gen2 architecture under [`doubleslash/`](https://github.com/doubleSlashde/umltheme/tree/main/doubleslash) |
| **Legacy compatibility** | Older include URLs remain available via redirects                                                                      |
